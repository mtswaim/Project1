# Stay-Wild

# Description
	1. Choose a pokemon
	2. Recieve a short list of details on selected Pokemon
	3. Enter Game with selected Pokemon 
  	4. Dodge Pokeballs

# API
	Pokemon API

# Features
	1. Hit API according to search based search
	2. Lets you choose any pokemon
	3. You get to use your Pokemon in the game
